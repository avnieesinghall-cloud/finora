# 🚀 Finora — AI Powered Personal Finance SaaS

A modern full-stack personal finance dashboard built using the MERN stack.

Finora helps users track income, expenses, budgeting, analytics, and financial insights through a clean SaaS-style interface with authentication and responsive design.

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/avnieesinghall-cloud/finora.git
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside backend:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
